Asset Tracking System

Vision Document

Project Mission

Produce a complete comprehensive Assets records and views at any time. These records can be modified.

Customers and benefits

Primary

Those companies who want exact record of Assets. This project can provide comprehensive assets records and views at any time. These records can be modified. 

Secondary

An alternative by making back up every week.

Key factors used to judge quality

Asset Tracking system must be interesting and easy to use for new and mature users.
The act of using must be so fun and realistic in itself that it attracts novice companies to manage their assets.

Key features and technology

  • Enter asset reports by employees.
  • View asset reports by Employees.
  • Enter other Information
  • View other Information
  • Preview Reports
  • Change Switchboard Items
  • Exit the database
  • Platform independent

  • Has to store on one Computer DRIVE
  • Must be extensible such that a later release can add to this DataBase
